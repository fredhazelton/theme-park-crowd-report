# Theme Park Crowd Report — Twitch Stream Vision
*Captured: 2026-01-28*

## The Concept

A **live data show** where Fred uses the pipeline dashboard to discuss, analyze, and answer questions about theme park wait times and crowds.

---

## Core Content

### Live Wait Time Analysis
- View actual wait time curves observed on current day
- Live wait times as they come in (posted times)
- **Key insight for viewers:** Posted times are overestimates with buffers — actual wait is usually shorter
- Model converts posted → actual times in real-time

### WTI (Wait Time Index)
- Present the Crowd Report using WTI
- Explain what it represents and how it's calculated
- Show: past days, current day, future forecasts

### Interactive Q&A
- Viewers ask about dates they're planning to visit
- Fred adjusts dashboard live to look up specific dates
- Discuss crowd patterns, best times to go, what to expect
- Historical analysis on request

## Key Differentiators

1. **Posted vs Actual** — Most people don't know posted times are inflated. This is insider knowledge.
2. **WTI** — A single number that summarizes crowd level. Easy for viewers to understand.
3. **Interactive** — Not just watching, but asking questions and getting personalized answers.
4. **24/7** — Even passive viewing builds the brand and community.
